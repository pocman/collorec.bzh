# collorec.bzh — Instructions pour Claude

## Langue

**Tout le contenu visible de ce site est en français.** Respecter scrupuleusement les règles suivantes :

- Écrire tous les accents correctement : é è ê ë à â ù û ü î ï ô œ æ ç
- Ne jamais laisser un mot sans accent quand il en requiert un (ex. : `à`, `é`, `è`, pas `a`, `e`)
- Respecter les règles typographiques françaises :
  - Espace insécable avant les signes doubles (`: ; ? ! »`) et après `«`
  - Majuscules accentuées : `É`, `À`, `Ê`, etc.
- Les titres, descriptions, boutons, libellés et messages d'erreur doivent tous être en français

## Stack

Site statique [Docusaurus](https://docusaurus.io/) (React/TypeScript), hébergé sur GitHub Pages.

- Pages : `website/src/pages/`
- Composants : `website/src/components/`
- Blog : `website/blog/`
- Données agenda : `website/src/data/events.json`
- Configuration : `website/docusaurus.config.ts`
